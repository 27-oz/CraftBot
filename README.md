# CraftBot — Discord Bot for Minecraft Communities

CraftBot is a quality-of-life Discord bot built with Python and discord.py.

## Features
- Welcome and goodbye messages
- Auto-roles on join
- Reaction roles
- Reminders and scheduled messages
- Custom commands and aliases
- Server stats and user info
- Polls with multiple options or yes/no
- TikTok post alerts via RSS
- Twitch live alerts via Twitch API
